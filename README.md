# DERIVATIVES PRICING LIBRARY

This project brings together all my work on options pricing into a single, structured library. Each module started as a standalone script, built while learning stochastic calculus, implementing formulas from my thesis, and experimenting with real market data. I reorganized everything into a proper Python package so the code is reusable, readable, and easy to extend.

---

## WHAT I WORKED ON

- **Black-Scholes**: Analytical pricing for European calls and puts, with full Greeks (Δ, Γ, ν, Θ, ρ) and implied volatility inversion via Brent's method.
- **Heston model**: Extended Black-Scholes by modelling volatility as a mean-reverting stochastic process. Implemented both Monte Carlo simulation (Numba-accelerated) and the analytical formula via numerical integration of the characteristic function.
- **Lookback options**: Pricing for floating-strike lookback calls and puts; both the closed-form formula (derived in my thesis) and a Monte Carlo engine with batch vectorisation.
- **Implied volatility surface**: Built the full IV surface by pricing a grid of options under Heston and inverting each price to an implied volatility via Black-Scholes. Shows the volatility skew generated by the negative price-variance correlation.
- **Risk metrics**: Sharpe ratio with 95% confidence intervals (Jobson-Korkie SE), drawdown tracking, equity curves, and real market data via yfinance.

---

## PROJECT STRUCTURE

- `black_scholes.py`: Analytical BS pricing, full Greeks, and implied volatility.
- `heston.py`: Heston model — Monte Carlo simulation, analytical pricing, and Greeks via finite differences.
- `lookback.py`: Floating-strike lookback options — closed-form and Monte Carlo.
- `surface.py`: Implied volatility surface from Heston prices, with 3D visualisation.
- `utils.py`: Sharpe ratio, drawdown, equity curve, and market data helpers.

## QUICK START

The following examples show how to import and use each module.

```python
from derivatives_pricing import black_scholes as bs
from derivatives_pricing import heston
from derivatives_pricing import lookback

# Black-Scholes Greeks
g = bs.greeks(S=100, K=100, T=1.0, r=0.05, sigma=0.20, option="call")

# Heston analytical price
price = heston.analytical_call_price(
    S0=100, v0=0.04, kappa=2.0, theta=0.04,
    xi=0.3, rho=-0.7, r=0.05, T=1.0, K=100
)

# Lookback put (closed-form)
price = lookback.analytical_put(S=100, S_max=100, r=0.05, sigma=0.20, T=1.0)

# Lookback put — Monte Carlo
price = lookback.mc_price(S0=100, r=0.05, sigma=0.20, T=1.0,
option="put", simulations=100_000)
```
